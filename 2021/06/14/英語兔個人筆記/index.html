    動詞語氣

    虛擬語氣(Subjunctive Mode)

    做「不可能」的假設

    張三有一天走在路上被兔子咬了一口只好住院，而這隻兔子已經逃到月球去了，此時躺在病床上的張三想著：

    「現在」沒發生但希望它發生：想象的假設、主觀的意願

    If I see the rabbit now, I will bite him. (X)
    If I saw the rabbit now, I would him. (O)

    If I knew it, I would tell you.
    If I were you, I would kill myself.

    「將來」發生的可能性極小

    萬一我明天看到那隻兔子，我要咬他。
    If I see the rabbit tomorrow, I will bite him. (X)
    If I should see the rabbit tomorrow, I would bite him. (O)

    If I should meet thee after long years, how should I greet thee?
    萬一我們將來重逢了，我又該如何面對你呢？ By 拜倫
    (咱倆相會是寥寥無期)

    與「過去」已發生事實相反: (過去完成)

    If I saw the rabbit few days ago, I would bite him. (X)
    If I had seen the rabbit a few dasy ago, I would have bitten him. (O)

    If I had watched this video before, I would have aced my test.

    歸納

    條件 If 主句
    現在與事實相反 V-ed would V
    將來不可能發生 should V would V
    過去與事實相法 had V-pp would have pp

    針對上述的句子發問(有些語法書會將他們歸類為疑問語氣)：
    Did the rabbit eat the carot?
    Was the rabbit hungry?
